SMS SPAM DETECTION SYSTEM

A SMS spam detection system built using Bidirectional Long Short-Term Memory (BI LSTM) is a type of machine learning model specifically designed to identify and filter out unwanted or unsolicited text messages (SMS) from legitimate ones.

The steps involves gathering a large dataset of SMS messages, comprising both spam and non-spam messages, preprocessing the collected data, training the model on the training data, evaluation of the model using testing data, then its deployment

HISTOPATHOLOGY GLAND SEGMENTATION

Histopathology image segmentation is a crucial task in medical image analysis, particularly in the field of digital pathology. It involves dividing a histopathology image into meaningful regions or segments to extract valuable information for diagnosis treatment planning.

Ensemble methods, which combine multiple models to improve performance, have been increasingly used in histopathology image segmentation due to their ability to enhance the robustness of the segmentation results
